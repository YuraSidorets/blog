Hi, I'm Yurii.

I work in software engineering and usually write about the parts of the job that are worth keeping notes on:

- .NET and backend design
- testing and debugging
- GitHub Actions and workflow automation
- architecture tradeoffs
- books and learning material that actually helped

This blog is intentionally small. The goal is not to publish polished thought leadership every week. It is to keep a useful set of technical notes, working examples, and opinions that came out of real engineering work.

## What you'll find here

Some posts are short notes. Some are implementation write-ups.

## Languages

The blog is bilingual:

- [English](/english/) is the default language
- [Українською](/ukrainian/) contains the Ukrainian posts and translations

When a post exists in both languages, there is a language switcher directly on the post page.

If you're new to the blog, [Start Here](/start-here/) is the quickest way to find the posts worth reading first.
